# My Exercism Solutions (C#) 🚀

В этом репозитории я храню свои решения задач с платформы [Exercism](https://exercism.org). 
Это часть моего ежедневного обучения для отработки синтаксиса и алгоритмов на языке C#.

## 🎯 Цели проекта
*   Ежедневная практика написания чистого кода.
*   Изучение стандартной библиотеки .NET.
*   Подготовка к техническим собеседованиям.

## 🛠 Технологии
*   **Язык:** C#
*   **Платформа:** .NET 7.0+
*   **Инструменты:** Exercism CLI, Visual Studio / VS Code

## 📂 Структура репозитория
Каждая папка соответствует конкретному упражнению. Внутри обычно находятся:
- `Solution.cs` — моё решение.
